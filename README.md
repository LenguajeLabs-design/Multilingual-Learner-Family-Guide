# WIDA Family Guide

**Created by Federico Orozco**

A multilingual web application designed to help families of English language learners understand their child's WIDA English proficiency level (1–6) and how to support their learning journey.

## About This Tool

The WIDA Family Guide helps EAL (English as an Additional Language) families:

- Understand WIDA proficiency levels 1–6
- See what skills their child is developing at each level
- Learn what supports are provided at school
- Find practical ways to help at home
- Understand how language development progresses over time

## Languages Supported

- English
- Spanish (Español)
- Chinese (中文)
- Korean (한국어)
